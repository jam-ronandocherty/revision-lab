30-minute session / Interleaved retrieval

Two topics, one session.

You enter key ideas for two different subjects, study both, then retrieve them together from a single recall effort. Mixing subjects during retrieval strengthens memory for each one more than studying them in separate sessions.

1

Set up 3–5 key ideas for two different topics.

2

Study both, then retrieve from memory in one attempt.

3

Fix the gaps per topic and plan when to return to each.

      08 — Plan

      Space out your returns.

      Retrieving each topic again after a gap builds stronger memory than re-reading it now. Choose a realistic time for each topic — they can be different.

      Why the gap matters: when partial forgetting happens before you retrieve again, your brain has to work harder. That extra effort strengthens the memory trace. Struggling slightly on the next attempt is a good sign, not a failure.
