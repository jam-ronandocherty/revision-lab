# Revision Lab

Revision Lab is a whole-school resource that teaches pupils how to revise effectively and then helps them put those methods into practice.

The planned resource is evidence-informed, framework-free, accessible and deployable as a static site on Cloudflare Pages.

## Initial product principles

- Teach revision through short practical experiences, not lists of tips.
- Centre retrieval, feedback, spacing and metacognitive reflection.
- Support different kinds of learning across the curriculum.
- Offer manageable 5, 15 and 30-minute revision sessions.
- Collect no pupil data; keep optional progress on the device.
- Work well on school laptops, tablets, phones and in print.

## Development

The site uses plain HTML, CSS and JavaScript. Open `index.html` directly or serve the folder with any static file server.
